# Análise de Dados: Nur Mah Museum

Este projeto consiste em uma análise técnica detalhada do setor cultural, com foco na exploração da base de dados do Nur Mah Museum. O objetivo principal é fornecer à direção executiva insights baseados em dados para fundamentar estratégias de captação de recursos e parcerias.

## 🎯 Objetivo do Projeto
Este projeto tem como objetivo analisar a base de dados dos museus para compreender melhor o cenário do setor cultural. A partir dessa análise, busca-se identificar padrões, tratar inconsistências nos dados e gerar informações que possam auxiliar na tomada de decisão do Nur Mah Museum.

## 🛠️ Tecnologias Utilizadas
- **Python:** Linguagem base para toda a análise.
- **Pandas:** Manipulação e tratamento de dados.
- **Seaborn/Matplotlib:** Visualização gráfica.
- **Jupyter Notebook:** Ambiente de desenvolvimento e documentação.

## 📂 Estrutura do Repositório
- `/docs`: Relatórios técnicos e documentos complementares.
- `/images`: Gráficos gerados durante a análise exploratória.
- `museum-analysis.ipynb`: Notebook principal com o código e a análise detalhada.
- `requirements.txt`: Dependências necessárias para reproduzir o ambiente.
